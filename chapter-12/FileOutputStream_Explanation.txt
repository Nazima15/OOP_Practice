파일 쓰기 (File Output)

파일 쓰기에는 바이트 출력 스트림 유형과 문자 출력 스트림 유형이 있음

바이트 출력 스트림 유형
- 응용 프로그램에서 출력 장치로 바이트 데이터를 전송하는 방법
- OutputStream 클래스: 출력(데이터 쓰기)을 하는 추상 클래스, 바이트 출력 스트림을 나타내는 모든 클래스의 슈퍼 클래스
- FileOutputStream 클래스: 파일에 데이터를 쓰는 데 사용되는 출력 스트림으로, 파일에 기본값(primitive value)을 쓸 때 사용
- FileOutputStream 클래스를 통해 바이트 지향 데이터와 문자 지향 데이터를 모두 작성할 수 있음

문자 출력 스트림 유형
- 응용 프로그램에서 출력 장치로 문자 데이터를 전송하는 방법
